# Exhausting A Crows


## Running Locally
```sh
$ npm install
$ npm start
```

The app should now be running on [localhost:5000](http://localhost:5000/).

## Deploying to Heroku

```
$ git push heroku master
$ heroku open
```

## Known Issues
- If your computer is connected to AirPlay, the plause function is delayed (in order to sync the audio)
